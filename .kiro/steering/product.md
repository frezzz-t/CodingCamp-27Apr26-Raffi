# Product Overview

**Expense & Budget Visualizer** is a client-side personal finance tracker that lets users log expenses, review their spending history, and visualize budget distribution by category through an interactive pie chart.

## Core Capabilities
- Add transactions with a name, amount, and category (Food, Transport, Fun)
- View a scrollable list of all recorded transactions
- Delete individual transactions
- See a running total balance updated in real time
- Visualize spending by category as a pie chart
- Persist all data across browser sessions via localStorage

## Key Constraints
- Runs entirely in the browser — no backend, no build step, no server required
- Openable directly as a local HTML file
- All data stored exclusively in `localStorage`; no external network requests
